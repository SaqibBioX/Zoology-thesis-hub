# Zoology Thesis Hub

A comprehensive web platform for zoology thesis resources, guides, and information.

## Features

- **Thesis Guides** - Step-by-step guidance for writing your thesis
- **Proposal Guide** - Help with thesis proposals
- **Resources** - Curated zoology research materials
- **Reference Database** - Access to important references
- **Tips & Best Practices** - Expert advice for thesis writing

## About

This is a thesis hub designed for zoology students and researchers in Nepal, providing all the resources and guidance needed to successfully complete a thesis project.

## Technologies Used

- React + TypeScript
- Vite
- Tailwind CSS
- Vercel (Hosting)

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- pnpm

### Installation

1. Clone the repository
```bash
git clone https://github.com/SaqibBioX/Zoology-thesis-hub.git
cd Zoology-thesis-hub
```

2. Install dependencies
```bash
pnpm install
```

3. Start development server
```bash
pnpm dev
```

4. Open http://localhost:3000 in your browser

### Building for Production
```bash
pnpm build
```

## Deployment

This project is deployed on Vercel. Any push to the main branch automatically deploys the site.
